Название проекта: DinoGameWithFeatures

Описание: DinoGameWithFeatures - это клон популярной игры "Dinosaur Game" от Google, написанный на языке программирования C#. Игрок должен управлять динозавром, прыгая через препятствия, которые появляются на его пути. Также добавлена возможность приседания динозаврика, что и является изюминкой новой версии игры.

Для создания игры использовался язык: C#

Чтобы запустить игру, следуйте этим инструкциям:

1) Склонируйте репозиторий на свой компьютер.
2) Откройте при помощи среды разработки проект.
3) Запустите проект.

Как играть: Управление игроком происходит с помощью пробела и ALT. Чтобы прыгнуть, нажмите на пробел, чтобы присесть - ALT.
